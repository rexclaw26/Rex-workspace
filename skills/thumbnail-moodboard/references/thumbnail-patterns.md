# Thumbnail Patterns Reference

## Proven Layout Patterns

### Pattern 1: "Big Face + Bold Text"
```
┌─────────────────────────────┐
│  [BOLD TEXT — 3-5 WORDS]    │
│                             │
│   [FACE — shocked/excited]  │
│                  [logo/icon]│
└─────────────────────────────┘
```
- Works best: opinion pieces, predictions, reaction videos
- Emotion: shock, excitement, disbelief
- Text position: top or bottom third, never covering face

---

### Pattern 2: "Split Screen"
```
┌──────────────┬──────────────┐
│  [FACE]      │  [GRAPHIC/   │
│              │   CHART/     │
│              │   LOGO]      │
│  [TEXT BELOW SPANNING FULL] │
└─────────────────────────────┘
```
- Works best: comparisons, analysis, "X vs Y" content
- Text: clear comparison label ("BTC vs ETH", "BULL vs BEAR")

---

### Pattern 3: "Data Reveal"
```
┌─────────────────────────────┐
│  [LARGE NUMBER OR STAT]     │
│  [context text — smaller]   │
│                             │
│  [face — bottom corner]     │
└─────────────────────────────┘
```
- Works best: price analysis, on-chain data, financial reports
- Number must be HUGE and immediately readable

---

### Pattern 4: "Controversy/Warning"
```
┌─────────────────────────────┐
│  ⚠️ [WARNING/SHOCKING TEXT]  │
│                             │
│  [face — alarmed]           │
│  [relevant logo/symbol]     │
└─────────────────────────────┘
```
- Works best: risk warnings, bearish takes, exposés
- Color: red accents, dark background with bright text

---

## Color Psychology for Crypto Content

| Color | Use When | Avoid When |
|-------|---------|-----------|
| Orange/Gold | Bull markets, gains, opportunity | Bear content |
| Red | Warnings, losses, urgency | Positive news |
| Green | Profits, growth, good news | Crisis content |
| Blue | Trust, education, analysis | High energy/hype |
| Purple | Premium, DeFi, innovation | Mainstream appeal |
| Black + neon | Tech/DeFi aesthetic, night content | Family-friendly |

## Text Rules

- Font weight: **Extra Bold / Black** minimum
- Font style: Sans-serif (readability over style)
- Text stroke: Dark outline on light text, light outline on dark text — always
- Max words: 5 (ideally 3)
- Numbers perform well: "$100K", "10X", "48 HOURS"

## Crypto-Specific Elements That Convert

- Bitcoin/ETH logos — immediately signals crypto content
- Price charts (especially dramatic ones)
- Dollar signs and price figures
- "Breaking" or "URGENT" labels (use sparingly — loses impact if overused)
- Recognizable faces: Musk, SBF, Coinbase logo, etc. (context-dependent)

## What to Avoid

- Too much text (more than 5 words)
- Dark background + dark text (no contrast)
- Cluttered backgrounds that compete with the face
- Generic stock images with no personality
- Fonts that are hard to read at small sizes (scripts, thin weights)
