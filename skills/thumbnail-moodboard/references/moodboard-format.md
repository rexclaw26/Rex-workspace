# Mood Board Format

## Mood Board Deliverable Structure

### 1. Direction Statement
One paragraph: What is the visual feeling we're going for? What should viewers feel when they see this brand/series/campaign?

### 2. Color Palette

| Role | Color | Hex | Usage |
|------|-------|-----|-------|
| Primary | [name] | #[hex] | Main brand color, CTAs, key text |
| Secondary | [name] | #[hex] | Supporting elements |
| Accent | [name] | #[hex] | Highlights, alerts, data callouts |
| Background | [name] | #[hex] | Page/slide backgrounds |
| Text | [name] | #[hex] | Body copy |
| Text on dark | [name] | #[hex] | Overlays, dark backgrounds |

### 3. Typography

| Role | Font | Weight | Size | Notes |
|------|------|--------|------|-------|
| Display / Headline | [font] | Black/900 | 48px+ | Video titles, thumbnails |
| Subheading | [font] | Bold/700 | 24-32px | Section headers |
| Body | [font] | Regular/400 | 16px | Readable body text |
| Data / Numbers | [font] | Bold/700 | Variable | Financial figures, stats |

### 4. Imagery Style
- **Photography:** [e.g., high contrast, editorial, authentic over stock]
- **Illustration:** [e.g., flat icons, 3D renders, data viz style]
- **Video/Motion:** [e.g., fast cuts, data overlays, broadcast-style graphics]

### 5. Layout Principles
- [e.g., asymmetric grids for energy, structured data-forward layouts for analysis content]
- [e.g., generous whitespace for premium feel, dense layout for news/urgency]

### 6. Reference Examples
- [Channel/brand name] — [what works, link if available]
- [Channel/brand name] — [what works, link if available]
