# Strategic Consulting & Brainstorming

**Trigger:** "Strategy session," "brainstorm," "McKinsey analysis," "strategic analysis," "growth strategy"

---

## 1. Strategic Analysis Framework

**Step-by-step approach for any strategic question:**

1. **Problem Framing** — Define the question, scope, and success criteria before touching the analysis
2. **Issue Tree / MECE Breakdown** — Decompose into mutually exclusive, collectively exhaustive components
3. **Hypothesis-Driven Analysis** — State a hypothesis first, then test it with data
4. **Competitive Benchmarking** — Compare against industry best practices and competitor moves
5. **Recommendation Matrix** — Score options on: Impact · Feasibility · Risk · Alignment with 90-day Rocks

See [frameworks.md](references/frameworks.md) for detailed methodology on each tool.

## 2. Brainstorming Sessions

**Ideation protocols:**
- Generate **10+ ideas before narrowing** — never self-censor during ideation
- Frameworks: First Principles · SCAMPER · Jobs-to-Be-Done · Blue Ocean Strategy
- Score every idea on four dimensions (1-10 each):
  - Revenue potential
  - Feasibility
  - Time-to-value
  - Strategic alignment with 90-day Rocks

**Deliver:** Top 3 recommendations with supporting rationale and concrete next steps.

See [frameworks.md](references/frameworks.md) for ideation framework details.

---

## 4. Output Standards

- **Lead with the recommendation** — support with data, keep it concise
- No 20-page reports — clear answer with the evidence behind it
- **Confidence level on every recommendation:** High / Medium / Low with rationale
- **Always include:** *"What would have to be true for this to fail?"*
- Apply Humanization Framework — REPORT mode (data-forward with personality)

See [output-templates.md](references/output-templates.md) for executive memo and recommendation matrix formats.

---

## Anti-Hallucination

Never fabricate market data, competitor intelligence, or trend information. Cite sources for all factual claims. If data isn't available, say so and frame the analysis around what is known.
