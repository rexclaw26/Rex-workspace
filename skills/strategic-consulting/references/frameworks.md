# Strategic Frameworks Reference

## Issue Tree / MECE

Break any problem into branches that are:
- **Mutually Exclusive** — no overlap between branches
- **Collectively Exhaustive** — together they cover the full problem space

```
Problem: "Why is revenue flat?"
├── Revenue drivers
│   ├── Volume (fewer deals?)
│   └── Pricing (lower rates?)
└── Revenue leaks
    ├── Churn (losing clients?)
    └── Payment delays (AR issue?)
```

Test: Can each branch be true/false independently? Do all branches together cover every possibility?

## First Principles

Strip away assumptions. Ask: what do we know to be fundamentally true?

1. Identify current assumptions about the problem
2. Break them down to basic truths (physics, economics, human behavior)
3. Rebuild from the ground up

**Hit Network application:** Don't assume crypto YouTube is the only model. What does the audience fundamentally want? What's the most direct path to deliver it?

## SCAMPER (Idea Generation)

Apply each lens to an existing product, process, or idea:

| Letter | Question |
|--------|---------|
| **S**ubstitute | What can be replaced? |
| **C**ombine | What can be merged? |
| **A**dapt | What can be modified or borrowed from elsewhere? |
| **M**odify/Magnify | What can be scaled up or emphasized? |
| **P**ut to other uses | What else could this be used for? |
| **E**liminate | What can be removed? |
| **R**everse/Rearrange | What if we flipped the order or approach? |

## Jobs-to-Be-Done (JTBD)

Focus on the outcome the customer is hiring the product to achieve.

**Template:** "When [situation], I want to [motivation], so I can [expected outcome]."

**Hit Network example:** "When I'm trying to understand a new DeFi protocol, I want clear, unbiased explanation from someone I trust, so I can make an informed decision without getting rekt."

Use JTBD to identify content angles, product features, and community gaps.

## Blue Ocean Strategy

Shift focus from competing in existing markets to creating uncontested space.

**Four Actions Framework:**

| Action | Question |
|--------|---------|
| **Eliminate** | What factors does the industry take for granted that should be eliminated? |
| **Reduce** | What should be reduced well below industry standard? |
| **Raise** | What should be raised well above industry standard? |
| **Create** | What should be created that the industry has never offered? |

## Competitive Benchmarking

1. Identify 3-5 relevant competitors or analogous companies
2. Compare on dimensions that matter (audience size, revenue model, content quality, engagement rate)
3. Identify gaps and opportunities
4. Flag sources for all data points

Never fabricate competitor metrics — use publicly available data or clearly mark as estimates.
