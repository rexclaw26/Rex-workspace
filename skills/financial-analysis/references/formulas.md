# Financial Formula Library

## Core Profitability

```
Gross Profit     = Revenue - COGS
Gross Margin%    = (Gross Profit / Revenue) × 100

Net Profit       = Revenue - Total Costs
Net Margin%      = (Net Profit / Revenue) × 100

ROI%             = (Revenue - Cost) / Cost × 100
ROAS             = Revenue Generated / Ad Spend
```

## Payback & Break-Even

```
Payback Period   = Initial Investment / Monthly Net Cash Flow

Break-Even Units = Fixed Costs / (Price per Unit - Variable Cost per Unit)
Break-Even Rev   = Fixed Costs / Gross Margin%
```

## Growth & Forecasting

```
MoM Growth%      = (This Month - Last Month) / Last Month × 100
YoY Growth%      = (This Year - Last Year) / Last Year × 100

CAGR             = (End Value / Start Value)^(1/Years) - 1

3-Month Moving Avg = (Month1 + Month2 + Month3) / 3
```

## Sponsor Deal Analysis

```
Deal ROI%        = (Sponsor Revenue - Fulfillment Cost) / Fulfillment Cost × 100
CPM              = (Cost / Impressions) × 1,000
CPC              = Cost / Clicks
CPL              = Cost / Leads Generated
```

## AI Operations Budget

```
Cost per Agent/Day    = Total API Spend / Active Agents / Days in Period
Token Efficiency      = Value Delivered / Tokens Consumed  [qualitative]
Budget Variance%      = (Actual - Budget) / Budget × 100
Over-Budget Threshold = >20% variance → flag to Kelly
```

## DeFi / Crypto

```
PnL              = Current Value - Cost Basis
PnL%             = (PnL / Cost Basis) × 100
Portfolio Weight = Position Value / Total Portfolio Value × 100
```
