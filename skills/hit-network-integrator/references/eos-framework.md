# EOS Framework — Templates & Patterns

Full scorecard and Rock tracking lives in the `weekly-scorecard` skill. Summary here for reference.

## Weekly Scorecard (Monday delivery)
5 sections: 90-Day Rock Progress · Key Metrics vs. last week · Issues (IDS format) · Wins · Priority Actions

## 90-Day Rocks
Track 3-5 priorities per quarter. Status: On Track / At Risk / Behind.
Current Rocks: see `weekly-scorecard/references/rocks-tracker.md`

## IDS Issue Format
**Issue:** [State clearly in one sentence]
**Impact:** [What happens if unresolved — specific]
**Recommendation:** [Proposed solution with data + confidence level]

## Integrator Task Queue Format
| Task | Status | Priority | Notes |
|------|--------|----------|-------|
| _(active tasks)_ | Active | High | _(context)_ |

## Operating Rhythm
- Monday: Weekly Scorecard + compliance audit
- Daily 7 AM: Systems health report
- Daily 7:30 AM: AI ops cost dashboard
- Daily 5 PM: Development summary
- Continuous: Monitor agent logs

See `weekly-scorecard` skill for full format and historical scorecards.
