# Article Self-Review Checklist

Run before delivering any draft to Kelly.

## Accuracy
- [ ] All facts verified — no fabricated statistics, quotes, or events
- [ ] All data points include `[Source: ...]` tags
- [ ] Price data pulled live from CoinGecko or CoinMarketCap
- [ ] Primary sources linked (on-chain data, announcements, filings)
- [ ] Financial disclaimer included if specific assets discussed

## Brand Voice
- [ ] Humanization Framework applied (correct output mode selected)
- [ ] Contractions used naturally throughout
- [ ] No robotic transitions (Furthermore, Additionally, Moreover banned)
- [ ] Sentence lengths vary — short punch, longer explanation, medium bridge
- [ ] No em dashes connecting thoughts
- [ ] Paragraphs 2-4 sentences max

## Structure & Readability
- [ ] Headline is compelling and specific
- [ ] Opening hooks immediately — no throat-clearing
- [ ] Subheads break up the body clearly
- [ ] Conclusion is strong, not just a summary
- [ ] Web-optimized readability throughout

## SEO
- [ ] Primary keyword in headline
- [ ] Primary keyword in at least one subhead
- [ ] Primary keyword in first paragraph
- [ ] Natural integration — not forced or repetitive
- [ ] Meta description written (155 characters max)
- [ ] 3-5 headline variations provided

## Delivery
- [ ] Outline was approved before drafting
- [ ] Length matches requested word count
- [ ] Publication destination formatting applied
