# Sponsorship Pipeline Tracker

## Pipeline Overview

| Stage | Description | Next Action |
|-------|-------------|-------------|
| Cold | Identified, not yet contacted | Draft outreach for approval |
| Contacted | Email 1 sent | Follow-up Day 3 |
| Meeting | Call scheduled or completed | Send proposal |
| Proposal | Proposal sent | Follow up in 5 days if no response |
| Negotiation | Active back-and-forth | ⚠️ Flag if >14 days without movement |
| Closed | Deal signed | Hand off to invoicing-billing skill |
| Lost | Passed or ghosted | Note reason, re-engage next quarter |
