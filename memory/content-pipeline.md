## X Rewrite Format Rules (confirmed 2026-03-18)
All 3 versions use the SAME structure: BREAKING/ALERT opener + ALL CAPS headline + -> bullets + closing question.
Variety comes from different hook angle and closer — NOT different formats.

- **V1:** Succinct. 2-3 bullets max, shorter headline, tighter. Same structure as V2.
- **V2:** Full version. 3-5 bullets, complete data story, crypto thesis bullet included.
- **V3:** Different hook angle. Different urgency opener if warranted (RED ALERT, JUST IN, 🚨 BIG WIN). Different closing question emotion/angle.

**Bullet rules:**
- Each -> line fits on ONE line, no wrapping (~60 chars max)
- Cut every filler word — every word earns its place
- Include a crypto thesis bullet when macro story warrants it ("Bitcoin's 21M cap doesn't budge")

**Closing rules:**
- Always 2 lines: one punchy statement + one rhetorical question on separate lines
- Targets audience identity/worldview — not generic curiosity
- Each version has a different angle: "Still waiting for the right time?", "Are you still on the sidelines?", "Are you still trusting the dollar?"
- Don't put crypto thesis in bullets AND the close — pick one. Let the close land clean.

**Source attribution rule:**
- When a credible source is available (Reuters, WSJ, Bloomberg), put it in the headline: "TRUMP CONSIDERS TROOPS per Reuters" — not buried in a bullet.

**Em dash rule:** Zero em dashes anywhere. Always "-" never "—". Em dash = AI fingerprint.
