### OpenClaw CLI Corrections (learned this session)
- `openclaw agents add main --provider openai` → WRONG flag, use `openclaw models auth paste-token --provider openai`
- `openclaw gateway token generate --scope read` → DOES NOT EXIST
- `openclaw config set operator.tokens [...]` → INVALID, "operator" is not a recognized config key
- Token generation: use `openssl rand -hex 32` to generate random tokens manually
- Scoped operator tokens: NOT supported in current OpenClaw version
